## Feature 2 — `feat/solo-mode-trigger` (sequence after Feature 1)

**What:** Heartbeat triggers solo mode on inactivity or explicit `wildwest.stepAway` command. Tier 1 auto-archival runs on each beat while solo. Configurable threshold (`wildwest.soloMode.inactivityThresholdMs`, default 15 min).

**Why sequence after Feature 1:** Solo mode autonomy tier depends on scope. Territory window = G/RA tier; town window = TM tier. Feature 1 must land first so solo mode can be scope-aware from the start.

**Implementation scope (for TM):**

- `SoloModeController.ts` — `enterSoloMode()`, `exitSoloMode()`, `executeTier1()` (archive `ack-done`/`ack-deferred` to `telegraph/history/`)

- `package.json` — `wildwest.soloMode.inactivityThresholdMs` config, `wildwest.stepAway` + `wildwest.returnFromAway` commands
