## Assignment 1 â€” Telegraph scripts (wildwest-framework/scripts/)

Write three canonical scripts. Copy-template to towns alongside `chat-end.sh` etc.

### `telegraph-send.sh`

Usage: `bash .wildwest/scripts/telegraph-send.sh`

Behavior:
- Run `date -u +"%Y%m%d-%H%MZ"` internally for timestamp â€” never accept as argument
- Prompt for: `to` (Role(Actor).Channel), `from` (Role(Actor).Channel), `type` (status-update|pr-draft-ready|ack-done|ack-blocked|ack-question|assignment|scope-change), `branch`, `subject` (slug, kebab-case)

- Print: "Created: <filename>"

**Filename rules:**
- Regular memo: `<ts>-to-<Role(Actor)>[.Channel]-from-<Role(Actor)>[.Channel]--<subject>.md`
- Ack memo: `<ts>-to-<Role(Actor)>[.Channel]-from-<Role(Actor)>[.Channel]--ack-<status>--<subject>.md`

### `telegraph-ack.sh`

Usage: `bash .wildwest/scripts/telegraph-ack.sh <original-memo-filename>`

Behavior:
- Accept original inbound memo filename as argument
- Parse `from` field (becomes ack `to`), `subject` from filename
- Prompt for: ack status (`done|deferred|blocked|question`), optional one-line question/note
- Run `date -u` for timestamp
- Write ack file with correct canonical filename
- Move original memo to `.wildwest/telegraph/history/`
- Print: "Acked: <ack-filename>; archived: <original>"

### `cleanup-telegraph.sh` (replace existing stub)
