## What's Changed

### Addressing Format (Role-Only)

**Old (deprecated v0.18.0, removed v0.19.0):**
```
to: CD(RSn).Cpt
```

**New (v0.18.0+):**
```
to: CD
```

The operator no longer cares about which devPair holds a role. Only the role matters.

### Town Pattern Matching

**New in v0.18.0:**
```
to: TM(*vscode)    # Route to any town matching *vscode
to: TM(*framework) # Route to any town matching *framework
```

Enables town-to-town memo delivery via wildcard patterns.
